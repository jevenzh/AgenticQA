# Waiver Control Rules

A waiver is a managed exception, not a silent relaxation of standards.

## Minimum Waiver Fields

- gate being waived
- approver
- expiry date
- rationale
- compensating control
- follow-up action owner

## Rules

- no waiver without an expiry date
- no waiver for unknown ownership
- repeated waivers on the same signal indicate a systemic gap, not a valid steady-state

## Theory and Practice Anchors

- regulated-quality and operational-governance practice: temporary deviations need bounded ownership and compensating controls.
- release governance best practice: unowned exceptions are hidden risk.
