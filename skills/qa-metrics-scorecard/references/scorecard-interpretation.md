# Scorecard Interpretation

A scorecard should explain what the numbers mean for release confidence.

## Interpretation Rules

- a green value with deteriorating trend is still a warning signal
- missing telemetry is a risk, not a neutral state
- one critical red metric can outweigh several green cosmetic metrics

## Theory and Practice Anchors

- decision-support reporting practice: metrics require interpretation and action context.
- release-readiness governance: confidence depends on risk concentration, not metric count.
